Typical errors: 

  • API not available

    This browser does not expose Magnetometer W3C Sensor API or at least not by default. On Android, it can be turned on by typing "chrome://flags" into the address box then set "Generic Sensor Extra Classes" to "Enabled". Other browsers/platforms will vary. 

  • Error: Could not connect to a sensor

    This browser exposes the Magnetometer API, but could not connect to an actual magnetometer. This is normal if running on a desktop computer which typically do not have an integrated magnetometer. Try on an Android phone or other mobile device.
